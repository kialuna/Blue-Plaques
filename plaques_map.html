Leeds Blue Plaques

The Leeds Blue Plaques Scheme

The Leeds Civic Trust have been installing blue plaques throughout Leeds since 1987. Their aim is to commemorate important people, events and buildings, and educate the public on the history Leeds. Blue plaques are nominated by members of the public, creating a sense of involvement and community for the residents of Leeds. More info on the scheme can be found on the Leeds Civic Trust website.

The Data

Blue Plaques data has been provided by The University of Leeds, and OSGB grid coordinates have been converted to WGS84 latitude and longitude using the JScoord library.

The Code

This map uses the Google Maps Javascript API. The code is based on code provided by The University of Leeds, together with this example of a clustered map with a sidebar.
